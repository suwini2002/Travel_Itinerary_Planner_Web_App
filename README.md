Sri Lanka Travel Itinerary Planner

A stunning, modern, and interactive web application designed to help users plan their dream vacation to the tear-drop island of Sri Lanka. Built with HTML, CSS, Bootstrap 5, and vanilla JavaScript.

Theme & Design
The user interface focuses on the rich beauty of Sri Lanka, employing a color palette inspired by its lush jungles (Jungle Green), golden beaches (Warm Golden Yellow), and deep oceans (Ocean Blue). 
The design uses:
- Glassmorphism for modern, semi-transparent overlays.
- Bootstrap 5 Card & Grid Layouts for a neat, mobile-first responsive design.
- Micro-interactions like hover scales, smooth transitions, and fixed scrolling navbars.
- Beautiful high-resolution imagery of top Sri Lankan tourist destinations.

Features
1. Home Page (`index.html`)
   - Striking Hero Section with a full-screen background image.
   - Interactive Image Carousel showcasing top tourist places (Kandy, Ella, Galle).
   - Featured Destinations Grid utilizing Bootstrap Cards.
   
2. Plan Trip Page (`plantrip.html`)
   - Dynamic Destination Adding: A fully functional JavaScript-driven form that allows users to add destinations to their itinerary.
   - Form Validation: Native and Bootstrap validation ensures that destination, date, and duration are provided.
   - Real-time Itinerary Summary: Calculates the total days and count of destinations on-the-fly.
   - Remove Destinations: Users can individually delete cards or clear the entire itinerary at once.

3. Contact Page (`contact.html`)
   - Contact form with client-side validation and simulated "sending" loading state.
   - Contact information cards.
   - Geographic visualization placeholder.

Usage Instructions
To view and interact with the project:
1. Clone or download this repository to your local machine.
2. Open `index.html` in any modern web browser (Google Chrome, Firefox, Safari, Edge).
3. Navigate through the site using the top navigation bar.
4. Go to Plan Trip and try adding a few destinations using the form to see the dynamic DOM manipulation in action.
